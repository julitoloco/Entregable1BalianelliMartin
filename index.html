FERRETERÍA CLEMENTE

Este sistema permite a los usuarios seleccionar una categoría de productos y visualizar los artículos disponibles dentro de ella.
Luego, pueden elegir un artículo específico y confirmar su selección. A continuación, se detallan sus principales funcionalidades:

  • Selección de categoría: Se solicita al usuario ingresar una categoría (Herramientas, Maquinarias o Electricidad).
  • Filtrado de artículos: Se muestran los productos que pertenecen a la categoría seleccionada.
  • Interacción con el usuario: Si hay artículos disponibles, se presenta una lista con sus nombres y números asociados.
  • Selección y confirmación: El usuario ingresa el número de un artículo para verlo en detalle. Se muestra su información (precio, marca, descripción) y se solicita confirmación.
  • Validaciones: Se manejan casos en los que la selección es inválida, la categoría no tiene productos o el usuario cancela la operación.
  • Filtrado adicional: Se incluyen funciones para listar artículos con stock disponible y aquellos con precio inferior a $1000. Este sistema garantiza una experiencia fluida en la selección y visualización de productos, incorporando validaciones para mejorar la usabilidad.
